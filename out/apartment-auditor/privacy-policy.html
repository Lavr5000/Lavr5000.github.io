Политика конфиденциальности

Apartment Auditor

1. Общие положения

Настоящая Политика конфиденциальности описывает порядок обработки данных в приложении «Аудитор квартир» (далее — Приложение). Разработчик Приложения — физическое лицо Сафин Д.А. (далее — Разработчик). Приложение предназначено для проведения технического осмотра и приёмки квартир в новостройках. Используя Приложение, вы соглашаетесь с условиями данной Политики.

2. Какие данные обрабатывает приложение

Приложение может обрабатывать следующие категории данных, вводимых пользователем:

  • ФИО, телефон, email, адрес участников приёмки
  • Паспортные данные: серия, номер, кем выдан, дата выдачи
  • Данные доверенности: номер, дата
  • Данные об объекте: адрес, номер квартиры, застройщик, ИНН/ОГРН, реквизиты договора
  • Фотографии дефектов, сделанные камерой устройства
  • Контактные данные, импортированные из телефонной книги устройства

3. Цель обработки данных

Данные обрабатываются исключительно для:

  • Документирования результатов технического осмотра квартиры
  • Фиксации выявленных дефектов с фотоматериалами
  • Формирования PDF-документов (актов осмотра, претензий, заявлений)

4. Хранение данных

Все данные хранятся исключительно локально на вашем устройстве (в хранилище AsyncStorage). Приложение не использует серверы, облачные хранилища или удалённые базы данных. Никакие данные не покидают ваше устройство автоматически.

5. Передача данных третьим лицам

Приложение не передаёт персональные данные третьим лицам. В приложении отсутствуют:

  • Системы аналитики и отслеживания
  • Рекламные модули

Приложение может отправлять анонимные отчёты о сбоях (текст ошибки и стек вызовов, без персональных данных) в закрытый канал разработчика для улучшения стабильности.

6. Использование камеры

Камера устройства используется исключительно для фотографирования дефектов при осмотре квартиры. Сделанные фотографии сохраняются только в локальном хранилище приложения и не передаются куда-либо.

7. Доступ к контактам

Приложение может запросить доступ к телефонной книге устройства для удобного импорта данных участников приёмки. Доступ используется только для чтения. Контактные данные не копируются, не сохраняются и не передаются за пределы приложения.

8. Безопасность данных

Защита данных обеспечивается средствами безопасности вашего устройства (блокировка экрана, шифрование хранилища). Приложение не реализует дополнительных механизмов шифрования, так как данные не покидают устройство.

9. Сроки хранения данных

Данные хранятся на устройстве до момента их удаления пользователем или удаления Приложения. Разработчик не имеет доступа к данным на устройстве пользователя.

10. Удаление данных

Вы можете удалить данные любого проекта непосредственно в приложении. При удалении приложения с устройства все данные удаляются полностью и безвозвратно.
